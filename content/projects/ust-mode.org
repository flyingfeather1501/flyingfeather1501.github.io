#+title: ust-mode

An Emacs major mode for editing UTAU projects (UST files). It is hosted [[https://github.com/kisaragi-hiu/org-msr][on Github]].

#+caption: Editing a UST
[[/emacs-ust-mode.png]]

Provides syntax highlighting (based on =conf-mode=) and a command to normalize paths like OutFile and CacheDir.
